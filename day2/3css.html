css란

CSS 는 HTML 문서의 스타일을 지정하는데 사용하는 언어입니다. CSS는 HTML 요소가 어떻게 표시되어야 하는지 설명합니다.

  • CSS 는 Cascading Style Sheets 의 약자입니다.
  • CSS는 한번에 여러 웹페이지(html파일)에 적용될 수 있습니다.
  • Style태그에서는 특성요소에 적용할 선택자(SELETOR)가 필요합니다. 참고로 style태그는 head안에 작성합니다.
  • CSS를 저장하는 파일 확장자는 .css입니다
  • CSS 작성은 태그에서 style 속성 사용하거나 또는 style 태그에 작성합니다
  • CSS 스타일의 프로퍼티는 글꼴 크기,색상, 배경이미지, 테두리 등등이 있습니다.모든 태그에 공통적인것 또는 특정태그에만 적용되는 것을 구별해서 사용합니다.
